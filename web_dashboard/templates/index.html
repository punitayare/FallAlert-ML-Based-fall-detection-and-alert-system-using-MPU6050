Getting Started

  1. Create a Virtual Device - Go to Devices page and click "Create New Device"
  2. Get Your Pairing Code - A 6-digit code will be generated (valid for 15 minutes)
  3. Start Virtual Simulator - Run the Python simulator script (Coming in Step 6)
  4. Monitor in Real-Time - Watch sensor data on the Monitoring page
  5. Test Fall Detection - Simulate a fall and receive alerts
